ackermann_mux
=========

Ackermann multiplexer with support for Ackermann topics and [std_msgs/Bool](http://docs.ros.org/api/std_msgs/html/msg/Bool.html) locks with priorities.

See [documentation](http://wiki.ros.org/ackermann_mux).
